Descripciones Generales

Tipos de Baby Arepas
Reina Pepiada: Es una arepa rellena de pollo guisado y luego horneada, acompañada de lonjas de aguacate y granos de petit-pois. Enlace para más info
Domino: Es una arepa la cual puede llegar a tener 3 tipos de combinaciones, con Caraotas, Porotos o Frijoles Negros, con queso, el cual le da una mezcla de salado y dulce perfecta. Enlace para más info
La Pelua: Es una arepa con relleno de carne mechada y queso amarillo rallado en hebras las cuales son las protagonistas de esta extraordinaria presentación. Enlace para más info
Tipos de Tequeños
Tequeños de Chocolate: Este tequeño es una preparación culinaria originaria de Medio Oriente que consiste en una masa de harina de trigo frita, rellena de chocolate, entre otros rellenos. Enlace para más info
Tequeños de Queso: También llamado dedito de queso, dedito, palito de queso, rollito de queso o tequeño está rellena de queso blanco o feta. Enlace para más info
Salsas a Añadir
Guasacaca: Se elabora principalmente con aguacate, perejil, pimiento, cilantro, cebolla, ajo y condimentos. Se usa como acompañante para carnes a la parrilla, pasteles, empanadas criollas, tequeños, pasapalos, etc. Enlace para más info
Salsa de ajo: Ésta no es más que una mayonesa casera diluida, con algunos ingredientes adicionales, bien cremosa y con bastante ajo. Es una receta simple y multifuncional. Enlace para más info
Salsa Runch: Es un condimento elaborado con suero de mantequilla o crema agria, mayonesa, cebollas verdes, ajo en polvo, y otros aliños mezclados en una salsa. Enlace para más info
Salsas con costo
Salsa blanca: Es una salsa cuyo principio de elaboración es el mismo que para la salsa bechamel, pero que incorpora un caldo, un fondo blanco o agua en vez de leche o nata. Enlace para más info
Salsa Picante: Ésta es rica en especias empleada frecuentemente como condimento, y en algunos casos como salsa para mojar. Enlace para más info
